# Ultrasonic-sensor
In this Arduino project, we are calculating the distance of an object from ultrasonic sensor. If the object is in the range of the ultrasonic sensor, then green bulb
glows otherwise red bulb glows.

## Components Used
1. Arduino UNO R3
2. Ultrasonic sensor
3. Red LED
4. Green LED
5. 1k ohm resistors
